Computers: How do they work?
Christopher PerezMay 29, 2021 at am

What is a computer?

When we think of computers, most of us think of a laptop or desktop computer. But in fact, laptops and desktops are just just two variations or formats that computers can have. Computers come in all shapes and sizes. Some can be very small, and be found in places like your smartwatch, other computers can be made up of dozens of other computers and work as one.

Computers have become ubiqoutous in our lives. Most of us simply turn on these devices and off we go(As they were intended), paying no mind to how these devices work even at a surface level.

Regardless of its form factor, a computer has four core parts; Input data, processor, memory, and data output.

Very often, you may find that there are many parts responsible for each of the aforementioned categories. These different parts are boud physically and through software to collectively achieve the task that they were meant to attain.

So what are these parts?

This article will take a look at some of the major parts in a computer that play a key role in ensuring that a computer functions as it is intended to. Please keep a notice that a computer is much more intricate than is described in this article. Along sections there will be links to explore more about certain topics.

Input data

Input data can be separated into two categories: external and internal. External input data refers to data that comes in from outside of the computer. Whereas internal refers to data from one part of a computer to another, such as memory to cpu(more later).

User data

Peripherals(keyboard, mouse, mic, and touchpad) Within the typical computer there are several devices responsible for capturing external input data. The most outstanding of these devices is the keyboard. The keyboard captures and translates the characters represented into its binary equivalent.

More on how keyboards work here. (https://www.explainthatstuff.com/computerkeyboards.html or my own article)

Another common device for input data capturing is the mouse/touchpad. Although different both devices work to help the user navigate the graphical user interface(GUI).

There is also the microphone. On modern desktop and laptops, microphones are integrated into the device and are meant to capture sound(analog data). Another part of the computer is responsible for translating the analog data into digital data that the computer can comprehend.

Lastly the camera. Similarly to the microphones, modern computers have cameras integrated. Cameras are useful for capturing the outside world.

Environmental Data

Not all external data comes from the user. Other specialized computers have sensors that capture environmental data such as atmospheric pressure, light, temperature, and all sorts of different conditions. Computers that have environmental sensors are usually not found in home environments and will most likely not look like your average user computer.

It is also important to keep in mind that data to a computers' processing unit can also come from other parts of the computer. Data that does not come from the user or other enviromental conditions to the computers' processing unit, can be consdired internal input data

Processing

The data processing part of the computer is handled by chips or the chipset(the collection of chips). Chips today are complicated but the underlying componenents still remain intact. Processors can be broken down into three parts at a moderate level of abstraction. The chip is composed of the arithmetic logic unit, control unit, and memory registers.

These parts work together to decode, compute and output data. Meaning that there are 3 instruction sets in one cycle.

The number of cycles that a CPU can make in one second is refered to as clock speed. Todays CPUs range in the gigahertz, meaning billion of cycles per second.

Processors are very complicated and delicate devices that are meant to perform instructions very fast. CPU's contain cache. Cache refers to memory that is made to be quickly accessed. The cache that is closest to the CPU(L1 cache) provides a way to rapidly access a small amount data, instead of requesting that data from its primary memory source.

There are three major chip architectures, serial, pipeline, and parallel. CPU's have a series of section through which data travels through before the CPU outputs some data. In a serial CPU, only one instruction is processed at a time. This means that there are parts of the CPU that are available for use. Pipeline processing takes advantage of this scenario by starting another instruction set before the one before it is done. This dramatically improves CPU performance. Pipeline processing can be thought of as a car assembly line. Lastly, as you might expect, parallel processing refers to executing multiple instructions at the same time. Parallel processing is feasible through multiple core processors. A core can be thought of as a chip within a collection of chips.

Modern CPU's have multiple cores that can range up to 16 cores. Higher end processors are used for demanding applications like gaming, machine learning, or high definition video editing.

Data output

Peripherals(monitor)

Once data has been processed the output is then display on the monitor or laptop screen. The primary form of data output is the screen monitor. There are several types of monitors found in user computers. Most often, monitors use a technology called liquid crystal display(LCD).

In LCD monitors light shines through liquid crystals to display an image on the screen. LCD monitors also use a technology called thin film transistor(FTF) which provides each pixel with its own transistor, thus effectively increase performance and color contrast. As mentioned before, LCD displays are backlit. There exist two backlighting variations: cathode flourescent lighting and light emitting diode(LED). Flourescent backlighting uses mercury vapor that when ionized emitts light. LED backlighting, in the other hand has a series of light emitting diodes that provide the back lighting technology. Moreover, LCD displays that have LED backlighting technology are called LED displays.

Memory

A key part of a computer that makes it usable is memory. Being able to store data for later use is important from both a user and processing perspective. There are many different types of memory that can be found in a computer.In the I.T. world, memory is defined as the location of short term data. while storage devices refers to the part of the computer that allows you store memory long-term.

In a computer memory can be broken down into primary and secondary memory. Primary memory is stored in volatile storage devices like DIMM and is called RAM(random access memory). While secondary storage is stored in non-volatile memory like magnetic-drives, SSD, or SSHD. Non-volatile memory acts as a long-term storage device that can hold its contents without a power supply.

Also, as mentioned before there is CPU cache. A small but very fast(and expensive) form of memory that holds information very close to the CPU to be accessed quickly. There are three levels of CPU cache: level 1, level 2, and level 3. Level 1 is the cache that is integrated in the CPU. L2 cache used to be memory that could be added to the processor, but now comes integrated in each processor core. L3 cache is cache that is shared by the processor cores.

You can find out more about Storage devices here(my article link or web link).

Power

Computers takes their foundations on sending data in binary across all the complex parts that are part of the computer. Electricity is required in order to send binary, but also to power the monitor, and give life to the volatile memory mediums like RAM. Without power there is no computer.

Power is provided by the power supply unit. The power supply unit is a term used to describe anything that provides power to the computer. In laptops, the power supply unit is the battery while in desktops is the power outlet(FACT CHECK STATEMENT).

All the essential components in the computer are bound together by the system board or motherboard. The motherboard carries within it electrical circuitry albeit pathways for electricity to travel from one device in the motherboard to another. In other words, the motherboard is the glue that connects all the devices integrated within it.

Device management

The motherboard ensures the interconnectedness of all the devices located within it. However, this fails to explain how everything is so precisely coordinated or why does one device knows how to send data to another. The second layer of connectivity is software and firmware. Software is code or instructions for the computer to execute. Programmers create code that is human readable and it then gets translated into a language that the computer understands(binary). The operating system and BIOS are two very special pieces of code that are responsible for making sure that everything works. It is thanks to these that you can turn on your computer and use your computer effortlessly(at least sometimes).

The Basic Input/Output System is a program that is stored in EPROM memory. During the boot up process the BIOS is responsible loading the operating system to the RAM and determining which peripherals are connected to the computer. Furthermore, the BIOS manages comunication between peripherals and the operating system. Later on, the BIOS was replaced by EUFI, which in essence is a more sophisticated version of the BIOS.

There are several popular operating systems like Windows, MacOS, Linux and linux distributions, IOS, and Android. The role of the operating system is to comminicate with the hardware to execute programs. Operating systems like the MacOS and IOS are hardware-specific operating systems(and proprietary). This means that they are built specifically for a certain hardware specs. On the other hand, with the help of drivers, Windows, Android, and Linux can run on just about any device. In fact, Linux(and its distributions) is the most common operating system in the world. Part of the reason is because linux is an open-source operating system that can be modified by the public. Hence, it is very popular because it can be modified to run on embedded devices that have very low resources. You can read more about operating systems and BIOS here.
