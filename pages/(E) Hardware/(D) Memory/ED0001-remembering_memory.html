Remembering Memory

 More often than not, components with electronics go bad. Fortunately, with some know-how servicing or replacing these components can be very easy (or hard). There are many hazards like electricity and fire that are present when working on systems even when they are off. Therefore, steps need to be taken and established to prevent and properly respond to any emergencies that can occur. Moreover, making sure that components are protected is also important when working on systems.

 Within the computer primary memory plays the central role of storing data for fast and short-term use. Thus enabling the computer hold and load applications or retreive data rapidly for fast computation by the computer. As opposed to retrieving data from the hard drive, a CPU retrieves its data from the RAM. This serves the purpose of being able to access data much more rapidly, along with preserving the hardrive by using it infrequently. Similary to the CPU, the size, speed, and type of RAM are strong dictating factors of the capabilities of a system. Having little RAM implies that the system cannot hold many applications at once without compromising speed whereas larger RAM sizes allow for the use of numerous applications and data.

 From a technical perspective RAM(Random Access Memory) refers to a volatile memory device that losses its contents when power is turned off. RAM uses special types integrated circuits in order to store data as ones' and zeroes. Memory speed is measured in Megahertz--Millions of cycles per second--, while the speed of legacy memory types is measured in the number of nanoseconds--billions of a second--it takes for data to reach the CPU.

 There are several different types of RAMs that exist, along with different form factors, and packages available.

DRAM

 The most common type of RAM used today is Dynamic-RAM(DRAM)--DDR More specifically--. DRAM uses capacitors as a means to store data. These capacitors cannot hold their contents for extended periods of time(leak) and therefore need to be refreshed periodically, hence the name dynamic. There are many flavors of DRAM available, each with distinct features that make them unique.


SDRAM

 Synchronous Dynamic RAM is a type of DRAM that runs at synchronous speed with the motherboard speed. Whereas other types of DRAM run at their specified speed, DRAM has the capability to match the speed of the motherboard is mounted on. Running synchronously with the motherboard means that data will be refreshed and accessed at the system speed. Although a great solution for a SDRAM that runs slower than the motherboard, the downside is that if you purchase a SDRAM that runs faster than the system, its speed will be truncated to match the system speed.

RAMBUS RAM

 Rambus RAM is a lesser known memory technology that served as the premium alternative to the compteting contemporary memory. Rambus RAM ran at 800MHZ on a 16-bit bus width(standard today is 64-bit). In comparison to the available alternatives at the time Rambus DRAM was much more advanced and destined to become the standard memory type in computer systems. However, due to lack of advancement in technology and cost Rambus RAM turned into an expense whose price was unjustified. Rambus DRAM was discountinued in the early 2000s after being replaced by DDR in terms of price and speed.

DDR

 Double Data Rate(RAM) is the current standard in memory technology. DDR RAM gets its name from being able to transmit twice the amount data per clock cycle. Over the years DDR has evolved to provide three new versions, all of which multiply the maximum speed of its predecestor. In addition, each succesive version also becomes much more power-efficient.

DDR2

 The second generation of DDR memory(DDR2) runs comes with varying speeds, the max which is 800MHz per second. DDR2 is much more power efficient than the previous generation by running at 1.8 volts as opposed to 2.5 volts for the original DDR.

DDR3

 In comparison to DDR2, DDR3 is stated to reduce power consumption by 30% while also increasing the capacity to a single(DIMM) module to 16GB.

DD4

 Lastly, the latest version of DDR memory is DDR4. DDR4 memory has an increased capacity per DIMM module of 64GB coupled with reduced power consumption. Along with all these new improvements also come a hefty price tag. DDR4 is by far much more expensive than its predecestors and is mainly used for computation heavy systems such as high end servers and premium gaming PCs.

SRAM

 Another type of RAM is Static RAM(SRAM). SRAM is much more faster in relation to DDR memory and it's therefore a lot more expensive than DDR memory. How much more expensive? A single GB of RAM can go for __Complete__me__. And how much faster? In nanoseconds, the amount of time it takes for SRAM to travel to the CPU ranges from 7 to 20 nanoseconds, while 50 to 70 nanoseconds for DRAM. SRAM is used in minute amounts on systems as memory for the CPU(cache) due to its speed. Unlike DRAM, SRAM can preserve its contents very well and does not need to be refreshed often as a result. Thus, due to a very low refresh rate, SRAM requires much less power to be maintained.

VRAM

 Video RAM is a purpose built RAM designed for video accelerator cards. in graphics cards VRAM is mainly used to store the value of pixels on the screen for screen refreshing purposes. A special feature that makes VRAM ideal for video is that it is dual-ported, meaning that it can send and receive data simultaneously.

CMOS RAM

 Complementary Metal Oxide Semiconductor RAM is another flavor of RAM that gets its name from the process it made. Comparitively, DRAM uses capacitors as a medium to store its contents while CMOS RAM uses Complementary Metal Oxide Semiconductors. In today's systems CMOS RAM is used to store the date and time so that the user does not have to manually input the date and time. Like other forms of RAM, CMOS RAM is volatile, so to preserve its contents the CMOS has its own battery attached. Previously, the CMOS RAM also served as an inventory list that maintained a list of the hardware devices and other system information. However, most contemporary systems have hardware information stored in the BIOS chip, which is a non-volatile memory type called EEPROM.

Packages

 Computers come in different shapes and sizes, and so does RAM. Packaging types refer to the differents form factors and pin distribution within a memory module. There exist four overarching types; SIMMs, DIMMs, SODIMMs, and MicroDIMMs.

SIMM

 Single Inline Memory Module(SIMM) is a legacy memory package that contained only one array of pins that supplied information in 32-bit chunks. The SIMMs form factor is designed for non-mobile systems like servers or desktop PCs. Memory must be installed in sizes that fill a memory bank(width of data channel), so in order to complete a 64-bit system, SIMMs are installed in pairs.

DIMM

 SIMMs are no longer used because they have been replaced by the Dual Inline Memory Module(DIMM). DIMMs contain to arrays of pins that provide information is 64-bits which is twice the amount provided in SIMMs.

SODIMM

 Small Outline Dual Inline Memory Module(SODIMM) is a memory package for laptops. SODIMMs come in different versions each with a differing number of pins for SDRAM and DDR memory. There exist a 32-bit 72-pin module, 64-bit 144-pin module for SDRAM, and a 64-bit 200 pin module for DDR and DDR2 memory.

MicroDIMM

 Micro Dual Inline Memory Module is an even smaller memory package suited for modern day laptops of low capacity. You can look at the different types of MicroDIMM available here.

Installing Memory

 An important detail to remember is compatibility when installing or upgrading memory. It goes without saying that the memory type you are installing should be supported by the system(i.e. install DDR memory in a DDR type system). When installing multiple memory modules make sure to check that the type and speed of the different modules match one another. Some memory types like SDRAM can be backwards and/or forwards compatible while others like DDR aren't. Typically, compatible memory types will having matching notches and pins to ensure the right types of modules are installed. However, there is nothing preventing you from purchasing incompatible memory types so is always a good idea to ask a representative if you're not sure.

 The same conceptuality applies for memory in mobile systems, although mistakes are less likely.
