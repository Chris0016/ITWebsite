What is the BIOS/UEFI?
Christopher PerezJune 19, 2021 at 6:30 pm

Understanding the BIOS/UEFI

 In a computer system, external devices such as RAM Modules, keyboards, or mice, need to communicate with the main system. The system also needs to identify the spefics of these devices to work with then efficiently. In order to solve the issue of connecting these devices with the system, the BIOS was made. BIOS stands for Basic Input-Output System, and it is essentially very low level code(or firmware) that aids in the integration and communication of these devices with the system. The BIOS has been around for over twenty-five years, which serves as a testament to its vitality in computer systems. Modern systems, however, use an enhanced version of the BIOS, namely the Unified Extensible Firmware Interface(UEFI). The UEFI provides extended capabilities and a richer graphical user interface(GUI). Lets take a look into the BIOS and UEFI.

 As mentioned before, at its core, the BIOS is firmware that allows for the integration and communication of upgradable devices on a system. The BIOS is stored EEPROM type chip. EEPROM stands for Electrically Ereaseable Programmable Memory, and it is a form of non-volatile memory that can be easily reprogrammed. Moreover, the BIOS has its own settings interface where changes can be made. Conventionally, the settings interface of the BIOS is called the BIOS Interface, but it is commonly refered to as the BIOS also.

 It is important to note that, information about external hardware devices is stored in CMOS RAM, and its settings are accessed through the BIOS interface in the 'CMOS Setup program' section.

 The BIOS/UEFI are important components of a system that indicate the capabilities of a system. In older systems, the BIOS had to be upgraded each time a new hardware change was made. The process of upgrading the BIOS was much more cumbersome on legacy systems since these chips used ROM technology and therefore needed to be completely replaced by a new physical chip. Luckily, most external devices today apply a concept called 'plug and play'. 'Plug and Play' means that the BIOS/UEFI can automatically detect changes on the system and make the necessary adjustments.

Upgrading the BIOS

 Of course not everyone has a modern system, so you may need to upgrade the BIOS. How to go about this? Very simple. As mentioned, to do an upgrade on older chips you need to simply run a special software. Using the version number/date you can get the software update from the manufacturer's website. The name of the manufacturer and version number/date are on the BIOS chip itself. Next, visit the manufacturer's website and download the appropriate BIOS update for the version of the BIOS chip that you have. Lastly, run the downloaded software, and done. Be sure to not interrupt the update process since it can render the computer to become dysfunctional.

UEFI

 The Unified Extensible Firmware Interface is the new and enhanced alternate to the BIOS/BIOS Interface. Over the years technology and methods to optimize technology have advanced very rapidly and so has the UEFI to accommodate for these changes. Some of the capabilities that the UEFI provides over the BIOS/BIOS Interface are: user-friendly GUI with mouse support, support for larger size disk size, improved security features, along with many more features that deal with current technology.
